# Falling Blocks — Мини-игра

Простая браузерная игра: управляйте прямоугольником и уклоняйтесь от падающих блоков.

Файлы:
- `index.html` — главная страница и холст
- `style.css` — стили
- `script.js` — логика игры

Запуск:
- Откройте `index.html` в браузере (двойной клик).

Либо запустите простой HTTP-сервер (если установлен Python):

```powershell
python -m http.server 8000
start http://localhost:8000
```

Управление:
- Влево/вправо: стрелки ← → или `A` / `D`
- Пауза/возобновление: `Пробел`

Если хотите, могу:
- добавить уровни или бонусы
- сделать адаптацию под мобильный экран
- переписать на TypeScript или добавить сборку

Скажите, что добавить дальше.
